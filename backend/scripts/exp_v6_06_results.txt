## Full Model

### Physiology Layer (Exp A/B/D)

PaO2_0 = 149.2 - PaCO2_0/0.8             [alveolar gas equation, Aa fixed at 0]

Apnea (t <= t_end):
  PaO2(t) = PvO2 + (PaO2_0 - PvO2) * exp(-t / tau_washout)

Recovery (t > t_end):
  PaO2_end = PvO2 + (PaO2_0 - PvO2) * exp(-t_end / tau_washout)
  PaO2(t) = P_EQ - (P_EQ - PaO2_end) * exp(-(t - t_end) / tau_reoxy)
  where P_EQ = 100 mmHg

CO2 (apnea): PaCO2(t) = PaCO2_0 + k_co2 * t
CO2 (recovery): PaCO2(t) = 40 + (PaCO2_end - 40) * exp(-(t - t_end) / tau_clear)
  tau_clear = 30s (fixed)

### ODC Layer (Severinghaus)

P50_eff(t) = 26.6 + 0.48 * (PaCO2(t) - 40)        [Bohr shift]
PaO2_virt = PaO2 * (26.6 / P50_eff)
PaO2_adj = 26.6 * (PaO2_virt / 26.6)^gamma          [steepness]
SaO2 = 100 / (1 + 23400 / (PaO2_adj^3 + 150*PaO2_adj))

### Sensor Layer — Smooth Discrete Gamma Kernel

Replaces the cascaded IIR from v6.05 with bin-integrated discrete gamma PDF.

v6.05 flaw: cascaded IIR used n_stages = min(round(1/cv^2), 12), which:
  1. Structurally clipped: any cv < 1/sqrt(12) ~ 0.289 -> identical 12-stage kernel
  2. Discontinuous objective: round(k) creates step changes in loss landscape

v6.06 fix: CDF bin integration via scipy.special.gammainc
  k = 1 / cv^2                           [continuous shape — no rounding]
  theta = mean_lag * cv^2                 [scale]
  h[i] = gammainc(k, (i+1)/theta) - gammainc(k, i/theta)    [bin-integrated]
  h = h / sum(h)                          [normalize]

Convolution with preconditioning:
  padded = [signal[0]] * len(h) ++ signal
  convolved = convolve(padded, h, mode="full")[:len(padded)]
  output = convolved[len(h):]             [strip padding]

Per-hold effective lag (Exp A: shift=0, Exp B/C/D: structural):
  effective_mean_lag_h = max(mean_lag + beta * g_h + delta_h, 1.0)

Output: SpO2_pred = clip(filtered + r_offset, 0, 100)

### Latent Spline (Exp C only — sensor-only identification)

Monotone PCHIP spline parameterized via decrements/increments:
  8 knots: [0, t_end/4, t_end/2, 3*t_end/4, t_end, t_end+10, t_end+30, t_end+45]
  Free params: x_0 (initial) + 4 decrements d_i (apnea, monotone down)
                              + 3 increments r_i (recovery, monotone up)
  Minimum is at t_end by construction (key separation assumption)
  PCHIP interpolation to 1-Hz grid, clipped to [0, 100]

Pipeline: latent spline -> gamma kernel convolution -> SpO2_pred

### Parameters

Exp A (17 params): 7 shared + 2x5 per-hold ICs
  PvO2        (15, 50)     Mixed venous PO2 (mmHg)
  k_co2       (0.02, 0.25) CO2 accumulation rate (mmHg/s)
  r_offset    (-8, 8)      SpO2 reading offset (%)
  tau_reoxy   (5, 30)      Re-oxygenation time constant (s)
  mean_lag    (5, 45)      Gamma kernel mean delay (s) [wider than v6.05's (5,35)]
  cv          (0.10, 1.2)  Kernel coefficient of variation [wider than v6.05's (0.15,1.0)]
  gamma       (0.8, 3.0)   ODC steepness exponent
  tau_washout varies       O2 washout time constant (s) [per-hold]
  PaCO2_0     varies       Initial arterial CO2 (mmHg) [per-hold]

Exp B (23 params): 7 shared + 1 beta + 5 delta + 2x5 per-hold ICs
  + beta      (-2, 2)      Depth-severity coefficient
  + delta_h   (-15, 15)    Residual per-hold shifts (narrower than v6.05's ±20)
  Constraint: lambda_zerosum * (sum delta)^2

Exp C (49 params): 4 kernel + 5 delta + 8x5 spline
  mean_lag    (5, 45)
  cv          (0.10, 1.2)
  r_offset    (-8, 8)
  beta        (-2, 2)
  delta_h     (-15, 15)    [5 per-hold residuals]
  x_0         (85, 100)    [per hold: initial SaO2]
  dec_i       (0, 20)      [4 per hold: apnea decrements]
  inc_i       (0, 30)      [3 per hold: recovery increments]

Exp D (19 params): 4 physiology + 5 delta + 2x5 per-hold ICs
  Frozen from Exp C: mean_lag, cv, r_offset, beta
  Fitted: pvo2, k_co2, tau_reoxy, gamma + 5 delta + 10 per-hold ICs

Fixed constants:
  Aa = 0 (A-a gradient, always at bound in v6.04 — removed)
  P50_BASE = 26.6 mmHg
  P_EQ = 100 mmHg
  tau_clear = 30s

### Loss Function

SSE = sum_i [ w_i * (SpO2_obs - SpO2_pred)^2 ]     on nadir window [0, t_end+45s]
  where w_i = 3.0 if SpO2 < 95%, else 1.0

Nadir timing:  lambda_nadir * (t_nadir_pred - t_nadir_obs)^2   [Exp A/B/D only]
CO2 priors:    lambda_k * (k_co2 - 0.06)^2 + lambda_paco2 * (paco2_0 - 40)^2
Gamma prior:   lambda_gamma * (gamma - 1.0)^2
mean_lag prior: lambda_mean * (mean_lag - 15.5)^2
r_offset prior: lambda_r * r_offset^2
IC reg:        lambda_reg * sum (IC_i - IC_type_mean)^2
[Exp B/C/D]:   lambda_shrink * sum(delta_i^2) + lambda_zerosum * (sum delta)^2
[Exp B/C/D]:   lambda_beta * beta^2
[Exp C]:       lambda_spline * sum(second_diff(knot_values)^2)

Regularization strengths:
  lambda_nadir = 1000, lambda_k = 2000, lambda_paco2 = 1000
  lambda_gamma = 2000, lambda_mean = 2000, lambda_r = 500
  lambda_reg = 10, lambda_shrink = 300, lambda_zerosum = 500
  lambda_beta = 200, lambda_spline = 50

Optimizer: scipy.optimize.differential_evolution
  Exp A/B/D: popsize=30, maxiter=3000, seed=42
  Exp C:     popsize=40, maxiter=5000, seed=42
  All: mutation=(0.5, 1.5), recombination=0.9, tol=1e-10, polish=True


## Changes from v6.05

1. Replace cascaded IIR with bin-integrated discrete gamma CDF (smooth, uncapped)
2. Wider bounds: mean_lag (5,45) was (5,35), cv (0.10,1.2) was (0.15,1.0)
3. Structural shifts: beta*g(severity) + delta_h with zero-sum constraint (Exp B)
4. Sensor-only identification with latent monotone PCHIP splines (Exp C)
5. Frozen sensor + physiology (Exp D)
6. No nadir timing penalty in Exp C (kernel must self-identify from shape alone)

## Analysis

### What worked

1. **Smooth kernel fixes the cv cap artifact.** v6.05's cv was constrained by the 12-stage
   IIR cap: any cv < 0.289 mapped to the same kernel. v6.06's bin-integrated kernel allows
   cv to reach 0.10 (Exp C) or settle at 0.213 (Exp A) without structural clipping.
   The objective is now smooth in both mean_lag and cv.

2. **Beta is consistently positive and interpretable.** Both Exp B (beta=1.39) and Exp C
   (beta=2.0, at bound) find that deeper desaturation events produce longer sensor lag.
   This is physiologically plausible: peripheral vasoconstriction during severe hypoxemia
   reduces blood flow to the fingertip, increasing pulse oximeter response time.

3. **Exp C's latent splines are physiologically plausible.** All start at 100%, show
   monotone decreasing SaO2 during apnea, with nadir depths matching expectations:
   FRC#2 (82%) < RV#3 (58%) < RV#4 (57%) < FRC#5 (47%). Recovery is minimal (0-2.5%),
   consistent with the short 45s window. The splines provide a credible "what SaO2 would
   look like without sensor distortion" picture.

4. **Exp C achieves the best R2 on nadir-window (0.891).** By fitting both the latent
   true signal and the sensor kernel simultaneously — without constraining to a physiology
   model — the sensor-only approach captures more data variation than the physiology-based
   experiments.

5. **Structural shifts reduce residual deltas.** In Exp C, beta absorbs nearly all
   per-hold variation: deltas are < 0.5s in magnitude. In Exp B, the residual deltas are
   smaller than v6.05's free deltas (max 15 vs 20).

### What didn't work

1. **Exp C kernel is at bounds (cv=0.10, beta=2.0).** The sensor-only experiment wants
   an even sharper kernel and even stronger depth-dependency than the bounds allow. This
   means the kernel is constrained, not truly self-identified. Widening beta bounds to
   (-5, 5) and cv to (0.05, 1.5) would test whether this is a bound artifact.

2. **mean_lag disagrees across experiments.** Exp A gives 22.5s, Exp B gives 16.0s,
   Exp C gives 15.8s. The 6.75s gap between A and B/C/D exceeds the 5s criterion.
   The difference comes from the model structure: without per-hold shifts, Exp A must
   use a larger global mean_lag to compromise across holds. This is a co-training artifact,
   not a kernel identification issue.

3. **Profile still monotonically decreasing.** Even with the smooth kernel fix (no cap,
   no rounding), loss decreases monotonically from mean_lag=5 to 45. At mean_lag >= 40,
   the optimizer compensates with unphysical parameters (k_co2=0.02 at bound, gamma=3.0
   at bound). The nadir penalty + mean_lag prior are insufficient to create a local minimum.
   This suggests the co-trained physiology+sensor model is fundamentally degenerate in
   the mean_lag direction.

4. **tau_reoxy at bound (30s) in ALL experiments.** Same structural issue as v6.04 and
   v6.05. The single-exponential recovery model is too simple.

5. **Exp D physiology params are unstable.** Freezing the sensor kernel from Exp C and
   fitting physiology gives pvo2=19.5 vs Exp A's 31.3 (11.8 difference). The frozen
   kernel was identified with cv=0.10 (near-delta function) and beta=2.0, which together
   produce a very different effective lag profile than Exp A's global kernel. The physiology
   must compensate, leading to very different PvO2 and timing parameters.

6. **FRC#2 remains problematic.** R2a is 0.21 (Exp A), -0.02 (Exp B), 0.63 (Exp C),
   0.26 (Exp D). Its moderate desaturation (78%) over a short hold (132s) provides
   insufficient signal for the model across all configurations.

### Key insights

1. **The data strongly prefers near-delta-function kernels.** When per-hold shifts are
   available (Exp B/C), cv drops to 0.10-0.11, making the kernel almost a pure delay.
   This confirms v6.05's finding: the sensor's temporal distortion is primarily a time
   shift, not temporal smearing.

2. **Beta captures a real physiological effect.** The consistent finding (beta > 0) across
   experiments suggests that pulse oximeter response time genuinely depends on desaturation
   severity, likely mediated by peripheral vasoconstriction.

3. **The co-trained model is degenerate in mean_lag.** Without external constraints (e.g.,
   step-response calibration data), the physiology model can always absorb changes in
   mean_lag by adjusting its own parameters. The profile sweep proves this: every mean_lag
   from 5-45s finds a plausible (if different) solution.

4. **Sensor-only identification is promising but under-constrained.** Exp C produces
   plausible splines and good R2, but the kernel hits bounds (cv, beta). Freezing those
   kernel parameters and fitting physiology (Exp D) produces worse R2 than co-training
   (Exp B), suggesting the frozen kernel is slightly miscalibrated.

5. **The two-stage pipeline (sensor-first, then physiology) needs additional constraints.**
   Step-response data, independent SaO2 measurements (e.g., arterial blood gas), or
   multi-session fitting could break the degeneracy.
